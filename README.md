# Social Media Platform (Django)

## Description
A social media backend built using Django and Django REST Framework.

## Features
- User authentication
- Posts & comments
- REST APIs
- Token-based login

## Technologies
- Python
- Django
- Django REST Framework
- SQLite

## Project Structure
users/
posts/
friends/
notifications/
api/
templates/
static/
tests/
docs/

## Run Project
pip install -r requirements.txt  
python manage.py migrate  
python manage.py runserver
